# Single price grid challenge

Desafio simples do Front-end mentor para desenvolver os conhecimentos utilizando grid layout



## Conhecimentos necessários

- HTML5
  - HTML semântico
- CSS3
  - Pré-processador SASS
  - Grid layout 
  - Flex layout
  - Layout responsivo
